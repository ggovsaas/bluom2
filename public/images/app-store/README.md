# App Store Images

This folder contains images for Apple App Store and Google Play Store submissions.

## Required Images

### Apple App Store (iOS)
- **App Icon**: 1024x1024px (required)
- **Screenshots**: 
  - iPhone 6.7" (1290x2796px) - 3-10 screenshots
  - iPhone 6.5" (1242x2688px) - 3-10 screenshots  
  - iPhone 5.5" (1242x2208px) - 3-10 screenshots
- **App Preview Videos**: 30 seconds max (optional but recommended)

### Google Play Store (Android)
- **App Icon**: 512x512px (required)
- **Feature Graphic**: 1024x500px (required)
- **Screenshots**: 
  - Phone: 320-3840px wide, 320-3840px tall
  - Tablet: 1080x1920px or larger
- **Promo Video**: YouTube link (optional)

## Image Guidelines

### Screenshots Should Show:
1. **Home Dashboard** - Daily stats, macro tracking, progress overview
2. **Fuel/Nutrition** - Food logging, macro breakdown, meal tracking
3. **Move/Workouts** - Exercise tracking, workout history, activity stats
4. **Wellness** - Sleep tracking, mood logging, habit tracking
5. **Premium Features** - AI food recognition, advanced analytics

### Design Requirements:
- Use actual app screenshots (not mockups)
- Show real data and realistic content
- Ensure text is readable at store sizes
- Use consistent branding and colors
- Highlight key features and benefits

### File Naming Convention:
- `icon-1024.png` (iOS app icon)
- `icon-512.png` (Android app icon)
- `feature-graphic-1024x500.png` (Android feature graphic)
- `screenshot-home-iphone.png`
- `screenshot-fuel-iphone.png`
- `screenshot-move-iphone.png`
- `screenshot-wellness-iphone.png`
- `screenshot-premium-iphone.png`
